# 🩺 Healthcare-risk-prediction-model

## 📌 Overview

The Healthcare risk prediction model is a machine learning-based application that predicts the likelihood of three major diseases:

* Heart Disease
* Diabetes
* Parkinson’s Disease

The system uses trained classification models on healthcare datasets and provides real-time predictions through an interactive web interface.

---

## 🚀 Features

* Predicts multiple diseases using a single platform
* User-friendly interface built with Streamlit
* Real-time prediction based on user input parameters
* Supports three different ML models for different diseases
* End-to-end pipeline: data preprocessing → model training → prediction

---

## 🛠️ Tech Stack

* **Programming Language:** Python
* **Libraries:** Pandas, NumPy, Scikit-learn
* **Visualization:** Matplotlib
* **Framework:** Streamlit
* **Dataset Source:** Kaggle

---

## 📊 Machine Learning Workflow

1. **Data Collection**

   * Imported healthcare datasets for heart disease, diabetes, and Parkinson’s

2. **Data Preprocessing**

   * Handled missing values
   * Feature selection and scaling

3. **Model Training**

   * Trained classification algorithms (e.g., Logistic Regression)
   * Separate models for each disease

4. **Model Evaluation**

   * Evaluated using accuracy score and validation techniques

5. **Prediction System**

   * Accepts user input
   * Outputs prediction results in real time

---

## 💻 Application Interface

* Built using Streamlit for simplicity and usability
* Users can input health-related parameters
* Displays prediction instantly

---

## 📈 Results

* Achieved ~85–90% accuracy across models
* Successfully predicts disease likelihood based on input features

---

## ⚠️ Limitations

* Accuracy depends on dataset quality
* Not a substitute for professional medical diagnosis
